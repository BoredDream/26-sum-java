## 通用规范

- **响应格式**: `Result<T>` = `{ code: int, message: string, data: T }`
- **分页格式**: `PageResult<T>` = `{ records: T[], total: long, pageNum: int, pageSize: int }`
- **状态码**: 200(成功), 400(参数错误), 401(未登录), 403(无权限), 404(不存在), 409(冲突), 500(服务器错误)
- **认证方式**: Session-based（登录后 `userId`/`role`/`name` 存入 HttpSession）
- **定时任务**: 每天凌晨2点自动备份数据库，保留30天
