# 信息管理模块 - 依赖表说明

## 一、模块管理的数据库表

### 1. notice（公告表）

| 字段 | 类型 | 说明 |
|------|------|------|
| notice_id | BIGINT PK AUTO_INCREMENT | 公告ID |
| title | VARCHAR(200) NOT NULL | 公告标题 |
| content | TEXT NOT NULL | 公告正文 |
| type | VARCHAR(50) NOT NULL | 公告分类（实训通知/截止提醒/系统通知） |
| attach_path | VARCHAR(500) | 附件存储路径 |
| top_flag | TINYINT DEFAULT 0 | 是否置顶（0否 1是） |
| publisher_id | BIGINT NOT NULL | 发布人ID（关联 teacher.teacher_id） |
| is_deleted | TINYINT DEFAULT 0 | 逻辑删除标记（0正常 1已删） |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
| delete_time | DATETIME | 删除时间 |

### 2. data_backup（数据备份表）

| 字段 | 类型 | 说明 |
|------|------|------|
| backup_id | BIGINT PK AUTO_INCREMENT | 备份ID |
| backup_time | DATETIME NOT NULL | 备份执行时间 |
| file_path | VARCHAR(500) NOT NULL | 备份文件存储路径 |
| file_size | VARCHAR(50) | 文件大小（可读格式） |
| operator_id | BIGINT NOT NULL | 操作人ID（关联 teacher.teacher_id） |

### 3. operate_log（操作日志表）

| 字段 | 类型 | 说明 |
|------|------|------|
| log_id | BIGINT PK AUTO_INCREMENT | 日志ID |
| operate_user_id | BIGINT NOT NULL | 操作人ID |
| operate_type | VARCHAR(50) NOT NULL | 操作类型（新增/修改/删除/备份/恢复/导入/导出/登录） |
| operate_content | TEXT | 操作详情描述 |
| operate_time | DATETIME NOT NULL | 操作时间 |

## 二、模块依赖的共享表（只读）

### 4. student（学生表）

| 字段 | 类型 | 说明 |
|------|------|------|
| student_id | BIGINT PK AUTO_INCREMENT | 学生ID |
| student_no | VARCHAR(50) UNIQUE NOT NULL | 学号 |
| student_name | VARCHAR(100) NOT NULL | 学生姓名 |
| major | VARCHAR(100) NOT NULL | 专业 |
| class_name | VARCHAR(100) NOT NULL | 班级 |
| phone | VARCHAR(20) | 手机号 |
| email | VARCHAR(100) | 邮箱 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 5. teacher（教师表）

| 字段 | 类型 | 说明 |
|------|------|------|
| teacher_id | BIGINT PK AUTO_INCREMENT | 教师ID |
| teacher_no | VARCHAR(50) UNIQUE NOT NULL | 工号 |
| teacher_name | VARCHAR(100) NOT NULL | 教师姓名 |
| office | VARCHAR(100) | 教研室 |
| title | VARCHAR(50) | 职称 |
| phone | VARCHAR(20) | 手机号 |
| email | VARCHAR(100) | 邮箱 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 6. user_account（用户账号表）- 核心认证表

| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | BIGINT PK AUTO_INCREMENT | 用户ID |
| username | VARCHAR(100) UNIQUE NOT NULL | 用户名（学号/工号） |
| password | VARCHAR(200) NOT NULL | BCrypt加密密码 |
| role | VARCHAR(20) NOT NULL | 角色（STUDENT/TEACHER/ADMIN） |
| related_id | BIGINT NOT NULL | 关联的学生/教师ID |
| status | TINYINT DEFAULT 1 | 账号状态（1正常 0禁用） |
| create_time | DATETIME | 创建时间 |

## 三、表关系

```
student ──1:1── user_account (role='STUDENT')
teacher ──1:1── user_account (role='TEACHER' or 'ADMIN')
notice  ──N:1── teacher (publisher_id → teacher_id)
data_backup ──N:1── teacher (operator_id → teacher_id)
operate_log ──N:1── student/teacher (operate_user_id → student_id or teacher_id)
```

## 四、索引建议

- notice: idx_type, idx_top_flag, idx_publisher_id, idx_is_deleted, idx_create_time
- operate_log: idx_operate_time, idx_operate_type, idx_operate_user_id
- data_backup: idx_backup_time
