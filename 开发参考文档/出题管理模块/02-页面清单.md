# 出题管理模块 页面清单

## 1. 模块名称

出题管理模块

## 2. 页面列表

| 编号 | 页面名称 | 页面路径 | 使用角色 | 页面内容 | 主要操作 |
|---|---|---|---|---|---|
| 1 | 题目列表页（教师） | `/topic/my-list` | 教师 | 显示当前教师本人创建的题目列表，含状态标签和筛选条件 | 查询、新增、查看详情、修改、提交审核、删除（草稿） |
| 2 | 新增题目页 | `/topic/create` | 教师、管理员 | 题目信息表单：项目名称、类型、难度、学生人数、开发工具、项目内容、技术路线、参考资料等 | 保存草稿、提交审核、取消返回 |
| 3 | 题目详情页 | `/topic/detail/:topicId` | 全部角色 | 显示题目完整信息、资料附件列表、审核状态/记录 | 查看、下载资料、返回 |
| 4 | 题目修改页 | `/topic/edit/:topicId` | 教师、管理员 | 预填题目现有信息的编辑表单 | 保存修改、重新提交审核、取消 |
| 5 | 资料管理页 | `/topic/:topicId/files` | 教师、管理员 | 题目相关资料文件列表，含文件名、大小、上传时间 | 上传文件、下载、删除、返回 |
| 6 | 题目审核页（管理员） | `/topic/review` | 管理员 | 待审核题目列表，含题目概要信息 | 查看详情、审核通过、退回修改、不予通过 |
| 7 | 审核操作弹窗 | （审核页内嵌） | 管理员 | 审核表单：审核结果（通过/退回/不通过）、审核意见 | 提交审核、取消 |
| 8 | 题库管理页 | `/topic/manage` | 管理员 | 全部题目列表，含全部状态标签和更多筛选条件 | 查询、开放、关闭、删除、查看详情 |
| 9 | 题目浏览页（学生） | `/topic/browse` | 学生 | 已审核通过且已开放的题目列表 | 查询、筛选、查看详情、下载资料 |

## 3. 页面之间的流程

### 教师流程
```
题目列表页 → 新增题目页 → 填写信息 → 保存草稿 / 提交审核
                                          ↓
                          提交审核 → 题目状态变为"待审核"
                                          ↓
题目列表页 → 修改(草稿/退回) → 题目修改页 → 保存/重新提交
题目列表页 → 查看详情 → 题目详情页 → 资料管理页 → 上传/下载/删除文件
```

### 管理员流程
```
题库管理页 → 查看全部题目
题目审核页 → 查看待审核列表 → 点击审核 → 审核弹窗 → 通过/退回/不通过
题库管理页 → 开放/关闭/删除题目
```

### 学生流程
```
题目浏览页 → 筛选查询 → 查看详情 → 下载资料
```

## 4. 需要统一的页面组件

| 组件 | 用途 | 说明 |
|---|---|---|
| 表格 | 展示题目列表、资料列表、审核记录 | 统一使用分页表格，支持排序和筛选 |
| 弹窗/抽屉 | 新增、编辑、审核操作 | 审核弹窗包含审核结果选择和审核意见输入 |
| 分页 | 列表分页 | 统一分页参数 pageNum/pageSize |
| 文件上传 | 资料上传 | 拖拽上传或点击选择，显示上传进度，限制格式和大小 |
| 状态标签 | 题目状态展示 | 不同状态使用不同颜色标签（草稿灰、待审核蓝、通过绿、退回橙、不通过红） |
| 搜索栏 | 筛选查询 | 关键词搜索 + 下拉筛选条件 + 搜索/重置按钮 |
| 二次确认弹窗 | 删除、关闭等重要操作 | 操作前弹出确认提示 |

## 5. 页面状态说明

### 题目审核状态标签

| 状态码 | 状态名称 | 标签颜色建议 |
|---|---|---|
| 0 | 草稿 | 灰色 |
| 1 | 待审核 | 蓝色 |
| 2 | 审核通过 | 绿色 |
| 3 | 退回修改 | 橙色 |
| 4 | 不予通过 | 红色 |

### 题目开放状态标签

| 状态码 | 状态名称 | 标签颜色建议 |
|---|---|---|
| 0 | 未开放 | 灰色 |
| 1 | 已开放 | 绿色 |
| 2 | 已关闭 | 红色 |
