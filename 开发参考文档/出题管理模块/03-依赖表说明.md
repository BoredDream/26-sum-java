# 出题管理模块 依赖表说明

## 1. 模块名称

出题管理模块

## 2. 本模块数据表

### 2.1 表名：`project_topic`（项目题目表）

| 项目 | 说明 |
|---|---|
| 用途 | 存储教师发布的综合实训项目题目信息 |
| 是否公共表 | 否，本模块专用 |
| 是否已确认 | 是，已进入 schema-v1.sql |

**核心字段：**

| 字段名 | 类型 | 说明 |
|---|---|---|
| `topic_id` | bigint | 主键，题目编号 |
| `topic_name` | varchar(100) | 项目题目名称 |
| `topic_type` | varchar(50) | 项目类型（如应用系统、管理系统等） |
| `difficulty` | varchar(20) | 项目难度（如 A/B/C 或 高/中/低） |
| `teacher_id` | bigint | 指导教师编号，关联 `teacher.teacher_id` |
| `student_limit` | int | 单个团队人数上限 |
| `selection_start_time` | datetime | 选题开始时间，可为空 |
| `selection_end_time` | datetime | 选题截止时间，可为空 |
| `team_limit` | int | 最大可选团队数，可为空表示不限制 |
| `topic_content` | text | 项目内容及要求 |
| `develop_tools` | varchar(200) | 开发工具与软件，可为空 |
| `technical_route` | text | 技术路线 |
| `status` | tinyint | 审核状态：0-草稿, 1-待审核, 2-审核通过, 3-退回修改, 4-不予通过 |
| `open_status` | tinyint | 开放状态：0-未开放, 1-已开放, 2-已关闭 |
| `is_deleted` | tinyint | 逻辑删除：0-否, 1-是 |
| `create_time` | datetime | 创建时间 |
| `update_time` | datetime | 更新时间 |
| `delete_time` | datetime | 删除时间 |

### 2.2 表名：`topic_file`（题目资料文件表）

| 项目 | 说明 |
|---|---|
| 用途 | 存储题目相关的项目指导书、任务书、参考资料等文件信息 |
| 是否公共表 | 否，本模块专用 |
| 是否已确认 | 是，已进入 schema-v1.sql |

**核心字段：**

| 字段名 | 类型 | 说明 |
|---|---|---|
| `file_id` | bigint | 主键，文件编号 |
| `topic_id` | bigint | 所属题目编号，关联 `project_topic.topic_id` |
| `file_name` | varchar(200) | 文件名称 |
| `file_path` | varchar(500) | 文件存储路径 |
| `file_type` | varchar(50) | 文件类型（如 docx, pdf, zip） |
| `file_size` | bigint | 文件大小（字节） |
| `upload_user_id` | bigint | 上传人账号编号，关联 `user_account.user_id` |
| `upload_time` | datetime | 上传时间 |
| `file_desc` | varchar(255) | 文件说明，可为空 |
| `is_deleted` | tinyint | 逻辑删除：0-否, 1-是 |

### 2.3 表名：`topic_review`（题目审核记录表）

| 项目 | 说明 |
|---|---|
| 用途 | 记录管理员对题目的每一次审核操作历史 |
| 是否公共表 | 否，本模块专用 |
| 是否已确认 | 是，已进入 schema-v1.sql |

**核心字段：**

| 字段名 | 类型 | 说明 |
|---|---|---|
| `review_id` | bigint | 主键，审核记录编号 |
| `topic_id` | bigint | 被审核题目编号，关联 `project_topic.topic_id` |
| `admin_id` | bigint | 审核人账号编号，关联 `user_account.user_id` |
| `review_result` | tinyint | 审核结果：1-通过, 2-退回修改, 3-不予通过 |
| `review_comment` | text | 审核意见 |
| `review_time` | datetime | 审核时间 |
| `create_time` | datetime | 创建时间 |

## 3. 依赖的外部数据表

| 表名 | 所属模块 | 用途 | 关联方式 |
|---|---|---|---|
| `teacher` | 信息管理模块 | 获取指导教师姓名/工号，校验教师身份 | `project_topic.teacher_id` → `teacher.teacher_id` |
| `user_account` | 信息管理模块 | 获取当前登录用户角色和关联ID，权限判断 | 通过 `user_id` 和 `role` 字段 |
| `student` | 信息管理模块 | 学生查看题目时校验学生身份 | `user_account.related_id` → `student.student_id` |
| `topic_selection` | 选题管理模块 | 判断题目是否已有学生选题（删除/修改保护） | `topic_selection.topic_id` → `project_topic.topic_id` |

## 4. 状态字典（本模块使用）

### 4.1 题目审核状态 `project_topic.status`

| 值 | 说明 | 说明 |
|---|---|---|
| 0 | 草稿 | 教师可自由修改、删除、提交 |
| 1 | 待审核 | 等待管理员审核，教师不可修改 |
| 2 | 审核通过 | 可被管理员开放 |
| 3 | 退回修改 | 教师可根据审核意见修改后重新提交 |
| 4 | 不予通过 | 终态，不可再次提交 |

### 4.2 题目开放状态 `project_topic.open_status`

| 值 | 说明 | 说明 |
|---|---|---|
| 0 | 未开放 | 默认状态，学生不可见 |
| 1 | 已开放 | 学生可见可选 |
| 2 | 已关闭 | 学生不可见，不再接受新选题 |

### 4.3 审核结果 `topic_review.review_result`

| 值 | 说明 |
|---|---|
| 1 | 通过 |
| 2 | 退回修改 |
| 3 | 不予通过 |

## 5. 表操作权限矩阵

| 操作 | 学生 | 教师（本人题目） | 教师（他人题目） | 管理员 |
|---|---|---|---|---|
| 查询 `project_topic` | 仅已开放通过的 | 本人全部 | 无 | 全部 |
| 新增 `project_topic` | 否 | 是 | — | 是 |
| 修改 `project_topic` | 否 | 仅草稿/退回 | 否 | 是 |
| 删除 `project_topic` | 否 | 否 | 否 | 是（逻辑删除） |
| 查看 `topic_file` | 仅已开放题目 | 本人题目 | 无 | 全部 |
| 新增 `topic_file` | 否 | 本人题目 | 否 | 是 |
| 删除 `topic_file` | 否 | 本人题目 | 否 | 是 |
| 查看 `topic_review` | 可见 | 可见本人题目 | 否 | 全部 |
| 新增 `topic_review` | 否 | 否 | 否 | 是 |
